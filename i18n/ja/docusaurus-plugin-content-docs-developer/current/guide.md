# Nodi開発者向けガイド

このドキュメントではNodiの開発に関わるために必要な情報を記述しています。

## コントリビューションについて

もし機能追加やバグ修正についてのコミットを行いたい場合、Github上でforkし、forkしたブランチ上でコードの編集を行ってください。

修正後に以下のコマンドでテストを実行できるので、テストがすべて成功すること確認してからpull requestを送ってください。

```bash

yarn test // jestによるテストを実行する

```
